Help & Tips

How to use:

  • Type any GTA V prop/model name in the input field
  • Click "Spawn Object" or press Enter to place it
  • Use the recent props buttons for quick access
  • Model names are case-sensitive

Examples:

  • prop_chair_01a - Basic chair
  • adder - Adder supercar
  • prop_tree_pine_01 - Pine tree
  • prop_container_01a - Shipping container

Tips:

  • Most props start with "prop_"
  • Vehicle names are usually just the model name
  • Check online GTA V model databases for more names
  • Invalid models will show an error message
